# 抽選アプリ

応募CSVをアップロードして、条件に応じた当選者・予備者を自動抽選するWebアプリです。

---

## 技術スタック

| 項目 | 内容 |
|------|------|
| フレームワーク | React (Vite) |
| CSV解析 | Papa Parse |
| デプロイ | AWS Amplify |

## 使い方（4ステップ）

### Step 1 - 応募CSVのアップロード
応募データのCSVファイルをアップロードします（ドラッグ＆ドロップ対応）。

### Step 2 - 抽選設定
以下の項目を設定します。

| 設定項目 | デフォルト値 | 説明 |
|----------|-------------|------|
| 当選数（U49） | 2名/日 | 49歳以下の1日あたり当選数 |
| 当選数（O50） | 3名/日 | 50歳以上の1日あたり当選数 |
| 予備数（U49/O50） | 同上 | 当選補欠の人数 |
| 対象都道府県 | 愛知県・三重県・岐阜県 | 絞り込む都道府県（カンマ区切り） |
| 年齢区分の境界 | 49歳 | U49/O50 の分割ライン |
| 基準年（年齢計算） | 2026年 | `基準年 - 生年` で年齢を算出 |
| 日別キーワード | 2026年3月各日分 | 応募日ごとに有効なキーワードでフィルタリング |
| 過去当選者CSV | 任意（複数可） | 除外対象の当選者データ |

### Step 3 - 抽選処理
処理中はローディング画面が表示されます。

### Step 4 - 結果の確認・ダウンロード
当選者・予備者の一覧を確認し、CSVとしてダウンロードできます。

## 抽選ロジックの詳細

### 1. データ正規化

CSVの各応募者に対して以下を計算します。

| フィールド | 内容 |
|-----------|------|
| `_appDate` | 応募日（YYYY-MM-DD形式に正規化） |
| `_name` | スペースを除去した名前 |
| `_age` | `基準年 - 生年` で算出した年齢 |
| `_ageGroup` | 49歳以下 → `U49`、50歳以上 → `O50` |
| `_personKey` | メールアドレス（個人の識別子） |
| `_householdKey` | `郵便番号_住所_建物名` の複合キー（世帯の識別子） |

住所・建物名は全角数字・漢数字を半角に変換して正規化します。

### 2. 重複排除・除外

- **同一人物の重複**: 同じメールアドレス + 同日の応募は1件に絞る
- **過去当選者の除外**: 以下のいずれかに該当する応募者を除外
  - 名前 + 世帯キーが過去当選者と一致
  - 世帯キーのみ一致（名前が異なっても除外 → 同居家族を含む世帯単位の管理）

### 3. 日付ごとの抽選

応募日ごとに以下の手順で抽選を実施します。

```
1. U49 / O50 のグループに分類
2. Fisher-Yates シャッフル（固定シードRNGを使用）
3. 上位N件 → 当選（当選）
4. 次のN件 → 予備（予備）
5. 当選・予備に選ばれた人は翌日以降の抽選から除外
```

### 4. 乱数生成（再現性の保証）

Linear Congruential Generator (LCG) にシード値 `42` を固定して使用します。

```javascript
function mkRng(seed) {
  let s = seed;
  return () => { s = (s * 16807) % 2147483647; return (s - 1) / 2147483646; };
}
```

**同じCSVを入力すれば、いつ実行しても必ず同じ結果が得られます。**

## 入力CSVのカラム仕様

### 必須カラム

| カラム名 | 内容 |
|---------|------|
| datetime | 応募日時 |
| name | 氏名 |
| email | メールアドレス |
| zip | 郵便番号 |
| prefecture | 都道府県 |
| address | 住所 |
| building | 建物名 |
| birthday | 生年月日 |
| keyword | 応募キーワード |

### 任意カラム（出力CSVに含められる）

`applicant_id`, `sex`, `city`, `phone`, `device`, `job`, `email_reject`

### 過去当選者CSV

| カラム名 | 内容 |
|---------|------|
| name | 氏名 |
| zip | 郵便番号 |
| address | 住所 |
| building | 建物名 |

---

## 出力結果

抽選結果のCSVには、元の応募データに加えて以下のフィールドが付与されます。

| フィールド | 内容 |
|-----------|------|
| `_result` | `当選` または `予備` |
| `_appDate` | 応募日 |
| `_ageGroup` | `U49` または `O50` |
